Introduction

Browser Flight Data Recorder is a firefox extension to automatically download web pages you visited. This extension takes snapshots of HTML when you leave web pages. Downloaded HTML files include form data, for example, search words in search engine page. Additionly, you can save each HTTP communication comprising HTTP headers and bodies of request and response. Snapshots of HTML and captured HTTP communications are downloaded automatically, without any clicks.

Format of saved file

A file of a snapshot of HTML is divided into two sections, Browoser Flight Data Recorder header and HTML content. Browoser Flight Data Recorder header includes meta information of the snapshot of HTML. HTML content is a snapshot of webpage. Two sections are divided by \n(new line character). A captured HTTP communication file id divided into five sections, Browoser Flight Data Recorder header, HTTP request header, HTTP request body, HTTP response header, and HTTP response body. Two sections are separated by \n (new line character).

Common fields
Field Description
version Version number is in X.Y.Z format, for example, 3.2.0. X stands for major verson, Y for minor version, Z for patch version. The major version increases when the compatibility of output files is lost. The minor version increases when new functions are released. The patch version increases when bugs are removed.
type If the file was generated when you leave a webpage (snapshot of HTML), type should be "OnBeforeDownload", while type is "HttpCommunication" if the log is a captured HTTP communication.
browserSessionId A session ID generated by Browser Flight Data Recorder. One session starts when one browser window is created and ends when all widnows are closed.
tabId A tab ID is a unique key in the same browserSessionId. When you open a tab, tab ID is generated. This tab ID survives until tab is closed.
Fields specific for HTML snapshot on you leave
Field Description
url This is the URL snapshot of web page you are leaving. This value may be different from that you requested to the server if web page used history.replaceState();
timestamp The Unix timestamp containing milliseconds at which you leave the web page.
Fields specific for captured HTTP communication
Field Description
requestUrl The destination URL of HTTP request.
requestTimestamp The Unix timestamp containing milliseconds at which HTTP request is issued.
requestMethod HTTP Method, for example, GET and POST.
requestOriginalUrl The original web page which requested HTTP communication. For example, if https://example.com has <script src="https://example.com/index.js"></script>, requestOriginalUrl field of HTTP communication to https://example.com/index.js is https://example.com. This field is empty if the HTTP communication is fired by parent HTML.
requestIp Destination IP address of HTTP communication.
requestProxyInfo The information of proxy. If you do not use proxy, this field is empty.
responseUrl The destination URL of HTTP request. This value is usually same as requestUrl.
responseTimestamp The Unix timestamp containing milliseconds at which HTTP response arrives.
responseMethod HTTP Method, for example, GET and POST. This value is usually same as requestMethod.
responseOriginalUrl The original web page which requested HTTP communication. This value is usually same as requestOriginalUrl.
responseIp The IP address from which response is received.
responseProxyInfo The information of proxy. If you do not use proxy, this field is empty. This value may be same as requestProxyInfo.
statusLine The status line in HTTP request.
fromCache true or false. This fields means whether the response is derived from the cache in browser or original server.
